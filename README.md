# 🎓 קורס הבנת כלי ה-AI - LMS

מערכת למידה אינטראקטיבית (LMS) מקצה לקצה לניהול קורס הבנת כלי בינה מלאכותית, פרומפטינג והנדסת אנוש מולם. המערכת מאפשרת לסטודנטים ללמוד את סילבוס הקורס, לתרגל מבחנים אמריקאיים ולקבל ציון ומשוב אוטומטי מיידי.

🚀 **האתר באוויר באקליק:** [כניסה למערכת ה-LMS שלך](https://prompt-academy-lms.onrender.com)

---

## 🛠️ טכנולוגיות וארכיטקטורה

المערכת בנויה בארכיטקטורת Full-Stack מודרנית וקלילה:
* **Frontend (ממשק משתמש):** HTML5, JavaScript (Async/Fetch API) ועיצוב רספונסיבי בעזרת **Tailwind CSS**.
* **Backend (שרת):** **Node.js** יחד עם פרימוורק **Express** לניהול נתיבים (API Routes) וטיפול ב-CORS.
* **Database (מסד נתונים קבוע):** אינטגרציה מלאה עם **Google Sheets** דרך ה-API של SheetDB לשמירה קבועה של ציוני הסטודנטים בזמן אמת.
* **Cloud Hosting (אירוח בענן):** השרת והאתר מאורחים ומנוהלים על פלטפורמת **Render** (שרת אירופאי בפרנקפורט לביצועים מהירים).

## 📚 סילבוס הקורס (14 מודולים מובנים)

| מודול | שם המודול | תיאור בקצרה |
| :---: | :--- | :--- |
| **1** | יסודות הבינה המלאכותית | מהו מודל שפה (LLM), איך AI חושב ועקרונות הזהב. |
| **2** | אנטומיה של פרומפט | עבודה עם Role, Context, Objective, Constraints ופורמטים. |
| **3** | טכניקות מקצועיות | תרגול Zero-shot, Few-shot, Chain Prompting ו-Meta. |
| **4** | מאסטר ב-ChatGPT | עבודה עם קבצים, מחקר, תמונות, Projects ו-GPTs מותאמים. |
| **5** | עבודה מתקדמת ב-Claude | שימוש ב-Projects, Artifacts, ניתוח מסמכים גדולים ו-MCP. |
| **6** | עוצמת Gemini | שילוב עמוק ב-Google Workspace, Drive, Docs ו-NotebookLM. |
| **7** | היכרות עם DeepSeek | יכולות, מגבלות, כתיבת קוד, לוגיקה והשוואה למודלים המערביים. |
| **8** | מטריצת השוואת מודלים | התאמת המודל הנכון למשימה הנכונה ותרגול פרומפטים במקביל. |
| **9** | AI בעולם האמיתי | יישומי מחקר, תכנות, ניתוח נתונים, אוטומציות ויצירת תוכן. |
| **10** | רמת Master & Workflows | בניית מערכות AI מורכבות, Multi-Agent והערכת איכות פלטים. |
| **11** | API - ממשקי תכנות ואינטגרציה | מהו API, איך מודלי שפה נחשפים כ-API, ומפתחות הרשאה. |
| **12** | MCP - Model Context Protocol | הפרוטוקול הפתוח שמחבר מודלי AI לכלים חיצוניים ומסדי נתונים. |
| **13** | Skills - יכולות ארוזות ומודולריות | מהי Skill, איך היא שונה מפרומפט רגיל, ומתי בונים אחת. |
| **14** | פלאגינים והרחבות (Plugins) | איך פלאגינים מרחיבים יכולות AI, וההבדל מ-MCP. |

## 📊 מבנה מסד הנתונים (Google Sheets)

הגשות הסטודנטים נרשמות באופן אוטומטי בגיליון אלקטרוני הכולל את העמודות הבאות:
1. `id` - מזהה ייחודי לכל הגשה (Timestamp).
2. `studentName` - שם הסטודנט כפי שהוזן במערכת.
3. `moduleNumber` - מספר המודול שתורגל.
4. `score` - הציון הסופי (מחושב אוטומטית בשרת).
5. `feedback` - פירוט התשובות הנכונות והשגויות.
6. `submittedAt` - תאריך ושעה מדויקים לפי שעון ישראל.
